# 🍬 Factory-to-Customer Shipping Route Efficiency Analysis

An interactive logistics analytics dashboard developed for **Nassau Candy Distributor** to analyze factory-to-customer shipping routes, optimize transportation efficiency, reduce logistics costs, and generate actionable business insights.

## 📌 Project Overview

This project provides an end-to-end visualization and analysis platform for shipping operations. It enables users to upload shipment datasets, explore delivery performance, analyze transportation costs, and identify opportunities to improve supply chain efficiency.

## ✨ Features

- 📊 Interactive Dashboard
- 🚚 Shipping Route Analysis
- 📍 Distance Analysis
- ⚡ Route Efficiency Analysis
- 🗺 Interactive Factory-to-Customer Map
- 🏭 Factory Performance Dashboard
- 👥 Customer Analysis
- 📈 Logistics Analytics
- 📄 Report Generation
- 📤 CSV & Excel Export
- 🌙 Dark/Light Mode
- 📂 CSV Dataset Upload
- 🔍 Search & Filtering

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/factory-to-customer-shipping-route-efficiency-analysis.git
```

Go to the project folder

```bash
cd factory-to-customer-shipping-route-efficiency-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
